# Exploratory Prose

Drop raw prose here — opening scenes, character sketches, anything you wrote
longhand to think on the page. These notes are **untagged**, so synthesis ignores them.
You harvest *sparks* out of them by hand (or with the Import prompt), then tag those sparks.
The prose stays here as a source you can revisit; it never feeds a canonical page directly.
