# Themes — motif bibles

One page per recurring theme in your story (trust, sins-of-the-father, hiding-yourself…).
Built like a Canonical note, but for a motif instead of an entity. Workflow:

1. Run the **Theme Discovery Prompt** (Copilot) over a scope of sparks — it reads for meaning
   and proposes candidate motifs + the sparks that embody each.
2. Approve the ones that are real (you decide; the AI only suggests).
3. Tag ONLY the confirmed sparks with the theme tag — namespaced, e.g. `theme/redemption`
   (no leading `#` in Properties). This is the one place theme tags get made — a tiny, earned set.
4. Make a page here from the Theme template; its Dataview block auto-collects the tagged
   sparks. Write your read of the motif and link the canonical pages it touches.

See the "Themes" section of MANUAL.md.
